            # 🪐Aurora Planetについて🪐

            Hello,world!!!
            こちらは、科学好きな日本語話者向けのFirefishサーバー、「Aurora Planet」です。当サーバーは以下の理念のもとに建てられました。

            • 様々なバックグラウンドを持つ方が科学に親しみ、楽しむことができる
            • 自分の専門分野だけでなく、他の分野の方との交流を後押しする
            • 一つの企業に振り回されないFediverseへの移行を応援する

            「Aurora Planet」をご利用の方は、以下のルールを守りましょう。 アカウントを登録した時点で、運営は、ユーザーがルールに同意したものとみなします。

            # 免責事項

            • Aurora Planetのサービスはベストエフォート方式です。
            • 通報などに対する対応はできるだけ速く(2日以内)に対応します。しかし、中の人の都合により、時間がかかる場合があります。
            • 分散型の仕様上、一度投稿した情報を完全に削除するのは困難です。
            • このサーバーは2023年6月からしばらくはクローズドテストを行い、2024年以降にオープンテストへ移行する予定です。
            • 運営は、サービス内で発生したユーザー同士のトラブルや不利益については、責任を負いません。
            • このサービスは仕様上、GDPR・UK GDPRの基準に対応できないため、EU加盟国とイギリスからのアカウント登録はできません。

            Due to the specifications of this service, it is not possible to register an account from EU member states and the United Kingdom, as it is not possible to comply with GDPR/UK GDPR standards.

            • このサーバーは私費で運営されています。いきなりサーバーダウンしたり、メンテナンスをする可能性があります。
            • サーバーを閉鎖するときは、閉鎖日の2週間前までに告知します。
            • 当サーバーのルールは必要に応じて随時更新されます。

            # 👤プライバシーについて👤

            • Aurora Planetの運営は以下の情報を収集します

              • アカウントの基本情報: パスワードとメールアドレスは非公開で、ユーザー名やプロフィール、avatar画像は公開で記録されます
              • 投稿・フォローなど、あなたが作成した情報:投稿やフォローなどは、公開する設定にすれば、公開されます。非公開設定のコンテンツや、チャットなどの非公開コンテンツも運営は閲覧できます。また、当サーバー外のユーザーとのやり取りの場合、あなたが意図しない人がこれらのコンテンツを閲覧する可能性もあります。運営や他の人に見られたくない情報は、投稿しないようにしましょう。
              • IPアドレス(アクセス情報):Aurora Planetは、当サービスにアクセスした時、IPアドレスなどの情報を収集します

            • Aurora Planetで収集された情報は以下の目的で使われ、それ以外の目的には使われません

              • Aurora Planetのサービスを提供するため
              • 何らかの理由でアカウントを利用できない時、該当ユーザーに連絡を取るため
              • モデレーションのため(アクセスが禁止されているかどうかを判定するために、情報を使うことがあります)
              • パスワードを忘れた時に送るリンク、その他メールでの通知のため

            • Aurora Planetで収集された情報は、下記の場合、同意なく第3者に提供することがあります

              • 日本国の法律や条例に基づく場合
              • 生命や財産の保護のために必要な場合で、本人の同意を得られないとき
              • 公衆衛生や児童の健全な育成の推進に必要な場合で、本人の同意を得られないとき
              • 法執行機関やその委託を受けた者が法律や条例の定める事務を遂行することに対して協力する必要がある場合で、本人の同意を得ることで、事務の遂行に支障が出るとき

            • Aurora Planetは、ソフトウェアのFirefishの性質上、Cookie🍪を使用しています。Cookie🍪はログインしたことを判別するために用いています

            # 📝Aurora Planetの投稿を使いたい方へ📝

            • Aurora Planetの投稿の著作権は、Renote/ブーストやリアクションなど他の人の投稿に関わるものを除いて、原則ユーザーにあります。

            • Aurora Planetの投稿を不特定多数の人が見られる制作物(例えば、公開のブログなど)に使用する場合は、トラブルを防ぐため、必ず本人に許可を取ってください。

            • それ以外の場合は、著作権法の範囲で自由にお使いください。
